# pygame_Snake
Projekt zrealizowany w ramach zaliczenia przedmiotu 'Analiza i projektowanie obiektowe'



# Instalacja

``` 
git clone https://github.com/2som/pygame_Snake
cd pygame_Snake
pip install -r requirements.txt
```

# Uruchomienie

```
python -m app.py
```


# Testy

```
pytest ./tests/
```
